# Humanly Staffing Enterprise Platform Requirements

## CRITICAL: This must be a fully functional enterprise staffing platform

### 1. VISUAL DESIGN
- Clone onhires.com hero section exactly with gradient backgrounds, stat counters, search bar overlay
- Match the overall enterprise aesthetic of onhires.com and wexpandmx.com
- Professional, polished look - NOT a basic template

### 2. IMAGES
- Unsplash/Pexels copyright-free images for team members
- Office scenes, professional workers
- Industry-specific imagery for each sector
- Hero background images matching onhires.com style

### 3. CLERK AUTHENTICATION
- Sign up / Sign in / Sign out flows
- Protected routes for candidates and employers
- Role-based access: candidate vs employer vs admin

### 4. DATABASE (Cognabase/Supabase)
- Jobs table with full CRUD operations
- Applications table linking candidates to jobs
- User profiles with resume uploads
- Employer company profiles

### 5. CANDIDATE PORTAL
- Dashboard showing all applied jobs with status
- Application history and tracking
- Profile management with resume upload
- Job recommendations based on profile
- Saved/bookmarked jobs feature

### 6. EMPLOYER PORTAL
- Post and manage job listings
- View and manage applications received
- Candidate search and filtering
- Company profile management

### 7. APPLICATION SYSTEM
- Fully functional Apply Now buttons that work
- Multi-step application form
- Resume upload and parsing
- Application confirmation emails

### 8. EMAIL INTEGRATION (MXRoute SMTP)
- Welcome email on signup
- Application confirmation to candidates
- New application notifications to admin
- Email drip sequences for candidates
- Weekly job digest emails

### 9. NOTIFICATIONS
- In-app notification center
- Real-time application status updates
- Email notification preferences

### 10. ADMIN DASHBOARD
- View all applications
- Manage users and employers
- Job approval workflow
- Analytics and reporting
